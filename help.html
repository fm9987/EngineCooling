Help using app

- Insert engine shape with x and r columns
- Check which kind of cooling you want
- Click the Data button on the top right
- Insert all the required data to run the simulation
- Once done press save and close window
- Run simulation with the button on the left
- After simulation is done you can check different outputs form the simulatino with the selectable items on the right
- A bar will appear under the engine that lets you select a point to highlight its data and will appear on the left
(On the bottom you can see messages from python for status of simulation)
(Restart button lets you start over)
(An excel file with output data will be saved as output.csv in same directory as shape file)
